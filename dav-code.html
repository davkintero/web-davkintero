Proyectos 🚀

Como futuro Arquitecto Frontend Developer cuento con diversos protyectos que validan mi eterno apredizaje y calidad en la ejecucion de los mismos. Implemento HTML, CSS y JavaScript puro; pero no me niego a usar Librerias, Framework o Procesadores si el modelo de Sitio Web e Interfas de Usuario lo requiere.

Les presento davCode un proyecto personal que esta en fase 1.0 y que he podido construir gracias a los cusos del gran senséi jonmircha. La versión 1.0 actualmente contine:

  • Una plantilla HTML bien estructurada con cabecera, área de contenido y pie.
  • Hoja de estilos enlasada al archivo html con el cual aplico: receteo, tipografía y cajas contenedoras Fluid Design basada en la filosofia de utopia.fyi, un diseño para cabecera con menu reposive, un diseño de contenido resposive y muchas cosas más.
  • Listado de clases con las propiedades de CSS Flex y CSS GRID para aplicar a los archivos html y poder construir diseño responsive.
  • Se aplica cuatro @media screen con tamaños (x=30em, m=48em, l=62em, g=75em) para adaptatar el viewport y poder visualizar mejor el diseño reponsive.
  • La clase dav aplicada a la etiqueta body en un archivo html, hace visualizar en la parte superios derecha un parrafo que nos dice el tamaño del viewport que esta trabajando. dav-style: Sinze (x, m, l, g), from (30em, 48em, 62em, 75em).
